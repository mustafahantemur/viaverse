# ADR-0001: Greenfield Platform Start

## Status

Accepted

## Context

Viaverse is starting from an empty repository. Existing React templates, prototypes, and screenshots are references, not implementation sources.

## Decision

Build Viaverse as a greenfield platform. Establish repository documentation, architecture guardrails, and development rules before adding backend or mobile code.

## Consequences

- Production architecture is not constrained by prototype code.
- Early work focuses on clarity, scope control, and shared language.
- Business flows are deferred until foundation decisions are stable.
